Input Format


The input contains three numbers A,B and C which denotes the angles of a triangle.

Output Format


Return the required result.

Constraints


0 ≤ A,B,C ≤ 180

Time Limit


1 second

Example


Sample Input

Sample Output
obtuse

CODE

const Triangle_Check = (A,B,C) => {
if(A<90 && B<90 && C<90){
return "acute";
}
else if(A>90 || B>90 || C>90){
return "obtuse";
}
};
